### 一、先搞懂：固定 Skill = 保住你 API 钱包的命根子

什么是固定 Skill？

说白了，就是把不需要动脑子的重复动作彻底锁死。

每次让大模型从头推理，等于你每次都花高薪请哈佛的谈判专家来帮你贴快递单——又贵、又蠢、还容易出错。如果你把“条件 A 满足就执行动作 B”的死流程写成 Skill，龙虾直接变成流水线上的无情熟练工：砍掉 90% 的无效上下文推理，API 消耗直接跳水，输出精准绝不跑偏。

这就是为什么同样是用龙虾，有人月费只要几十刀就能干翻全场，有人烧了几百刀还在群里骂娘。

### 二、真正的王炸：WhatsApp 直连 ERP（降维打击）

这才是龙虾目前的终极形态，也是拉开差距的核心护城河：把 WhatsApp AI 入口、龙虾、ERP 全链路彻底打通。

以前客户在 WhatsApp 问一句：“你们那个型号有没有库存？交期多久？MOQ 多少？FOB 价格多少？” 你的操作是：切出软件 → 登录 ERP → 查数据 → 掏出计算器算价格 → 切回 WhatsApp 回复。客户嫌慢，早就找别人去了。

现在的高端局玩法：客户在 WhatsApp 发问，龙虾直接通过 API 调用你的 ERP 数据库。秒查库存、交期、价格，甚至直接生成 PI 核心信息回传。零切换、零等待、零人工手动操作。

配合固定 Skill，龙虾在这里不推理、不瞎编、不烧无用 API。你用 WhatsApp 开发客户的响应速度和专业度，直接对同行形成降维打击。

### 三、这 3 件事，必须写死成 Skill，半分推理都别给！

除了打通 ERP，以下这些纯体力活，赶紧送进流水线：

#### 1. 询盘扒皮 &amp; 客户背调（0 推理，纯提取）

脑残用法： “帮我分析这个阿里的询盘怎么回？”（烧几千 token 换一堆正确的废话）。

老鸟 Skill 焊死法： “你是一个冷酷的外贸询盘提取器。只按以下 JSON 格式输出：1. 国家及推测时区；2. 客户类型（终端/贸易商）；3. 明确的 SKU/参数；4. 隐藏痛点（急交期/死命压价）。禁止任何客套废话！”

效果： 一秒钟把垃圾钓鱼邮件和高质量 VIP 分层，API 几乎零消耗。

#### 2. 标准化 PI &amp; 报价生成（无情填表机器）

把你们公司的阶梯报价公式、FOB 港口杂费、退税率全部写进 Prompt。

让 AI 只做代数计算和填表，绝对不准自由发挥产品参数。精准、快速、零出错。

#### 3. 报关单证核对（纯粹找茬，严丝合缝）

单证错一个字母，改单费和目的港罚款就能让你这单白干。

老鸟 Skill： “只交叉对比提单、商业发票、装箱单的这三项：1. 唛头是否完全一致；2. 毛净重、体积是否吻合；3. 收货人信息有无错别字。有错标红，没错只回‘单证一致’。”

效果： 比人眼盯十遍都稳，花最少的钱，避最大的坑。

### 四、写好 Skill 的底层逻辑：你越懂行，AI 越像老炮

90% 的人写不出能用的 Skill，根本原因是他自己对产品、行业、公司的底牌一窍不通。

记住，Skill 不是抒情散文，是工业说明书！比如你是卖数控机床的，千万别在指令里写“给客户报个好价”。你要写死：

“只要提及三轴加工中心，必须强调我们用的是台湾 PMI 导轨、发那科系统。如果客户嫌贵，立刻抛出换‘国产系统降价 15%’的备选方案。语气要像个干了 20 年的老工程师，不卑不亢。”

你给的业务约束越强，AI 越不会产生幻觉，不仅不跑偏，API 还贼省钱。

### 五、这些高端局，该烧的 API 一分也别省！

固定 Skill 省下来的钱，全给我砸在必须动脑子的高端流程上

深度逼单与价格拉锯战： 客户拿印度同行的低价压你，你需要让 AI 结合十轮历史邮件、市场关税变化做深度博弈分析。

大客户的高转化定制开发信： 结合客户官网新闻、领英动态找破冰点（Hook）。

客诉、货损等危机公关： 安抚情绪、撇清责任、给出方案，这需要极高的情商和逻辑。

这些场景每次都不同，必须全量推理。在这上面省 API，就是变相丢单子，纯纯的捡芝麻丢西瓜。
